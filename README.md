# MeridianLX Organization

Welcome to the **MeridianLX** GitHub organization! We are the team behind **Meridian Server OS** and related projects, dedicated to building a minimal, efficient, and modern Linux-based server platform.

## Projects
- **Meridian Server OS**: Minimal, production-ready Linux server distribution with a focus on simplicity and reliability.
- **Meridian Desk**: Optional, ultra-lightweight desktop environment for Meridian.
- **mpkg**: Custom Bash-based package manager for Meridian, supporting real downloads, dependencies, and custom repositories.
- **Buildroot Integration**: Automated scripts and configs for building custom Meridian images.
- **Website & Docs**: Modern, community-driven documentation and project website.

## Key Features
- Minimal base system (x86_64, glibc, BusyBox)
- Stable (SE) and Beta (BE) editions
- Optional desktop (Meridian Desk)
- Custom package manager (mpkg)
- Simple, text-based configuration
- Open source and community-driven
